# EMBER — the bug-out box

**Say "Hey Ember" and ask.** An offline emergency-preparedness AI in a box.
Raspberry Pi 5 + Hailo-10H NPU,
Gemma 4 E2B on-device, voice in / voice out, grounded in a local survival
corpus with citations. No cloud. No subscription. The internet being gone is
the use case.

Built at **Build with Gemma: JustBuild** (Google DeepMind / Pattern / Mindsmith),
July 17-18 2026 — Track 1: On-Device AI with Gemma 4.

Repo created at kickoff; all code written during the event.
Knowledge corpus (public-domain field manuals + offline Wikipedia medicine)
prepared in advance as data, documented in DATA.md.
